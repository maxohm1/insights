# Insights

Insights is a beautifully designed Android application focused on tracking personal health trends, body signals, and lifestyle impacts. The UI is built completely natively using modern Jetpack Compose.

## Features

- **Stability Summary:** A gorgeous layered area chart displaying an aggregated stability score over time with an interactive tooltip.
- **Cycle Trends:** Custom bar charts showing cycle lengths, complete with dynamically filled sections.
- **Body & Metabolic Trends:** Smooth cubic bezier line charts for weight tracking.
- **Symptom Trends:** A clean donut chart displaying precise symptom percentages (e.g. Bloating, Mood, Fatigue).
- **Lifestyle Impact:** A dynamic correlation matrix showcasing how Sleep, Hydration, Caffeine, and Exercise impact your overall stability.
